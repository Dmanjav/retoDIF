1. Configurar en mysql el usuario para la base de datos.
2. Configurar el idioma de mysql en español de la siguiente forma
    1. Abrir el archivo en la siguiente ruta:
        /etc/mysql/my.cnf
    2. Copiar las siguientes líneas al final del archivo
        [mysqld]
        lc_time_names = "es_MX"
    3. Reiniciar el servicio
        sudo service mysql restart

2. Iniciar sesión en mysql
    mysql -u <usuarioBD> -p
3. Utilizar los siguientes comandos para cargar la BD en mysql
    source <Ruta al archivo APPDIF_MARIADB.sql>
    source <Ruta al archivo APPDIF_MARIADB_DATA.sql>

4. En el servidor configurar las variables de entorno
    DB_DIF_USER = <Usuario de la base de datos>
    DB_DIF_PASS = <contraseña de la base de datos>

5. Configurar un entorno virtual de python dentro de la carpeta /web y activarlo
6. Ejecutar el siguiente comando para instalar las dependencias requeridas
    pip install -r <Ruta al archivo pip_requierements.txt>
